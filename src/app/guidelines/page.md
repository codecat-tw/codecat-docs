# 網頁開發指南

正因為程式貓使用了現在化的網頁架構，才得以同時維護龐大的服務。這篇指南詳細的講述了團隊使用的架構，提供你在最佳實踐道路上的方向。這個技術選擇同時作為程式貓及數個團隊的標準，是個經過生產測試的穩定方案。

當然，最好的架構往往也不足以適應所有場景，你仍然需要根據當下情況作出微調。

## 全端框架

- Next.js (使用 src 目錄)

## 部署服務

- Vercel
- Zeabur (若需考量成本時)

## API

- Next.js route (RESTful API)

## 資料庫

- PostgreSQL (生產環境)
- Firstore (MongoDB 可用於 PoC)

## ORM

- Prisma

## 身分驗證

- Better Auth

## 快取

- Tanstack Qurey

## CSS

- Tailwind CSS

## UI 框架

- Base UI
- Shadcn/UI (可選)

---

本指南會隨時更新以反映瞬息萬變的前端生態系統，歡迎隨時重新瀏覽此頁面。如果你正在開發一個網站或想要加入程式貓，熟悉這樣的架構將會很有幫助。
