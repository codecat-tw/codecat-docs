# 程式貓文件中心

這裡是程式貓科技教育社群開源的文件網站，用於提供程式教學、技術文件、社群資料統整等功能。網站開源可編輯，你隨時可以協助我們補充資訊或修復錯誤。

## 認識程式貓

程式貓是以科技教育為導向的專案型社群，奉行「讓科技教育走入生活」的核心理念。 有別於其他社群以單一計畫向社會提供服務，程式貓更加注重成員在建立產品過程中的學習與成長。 在數位化世代，我們相信每個人都有權利實現心中的專案，並藉此服務社會。 程式貓希望能陪伴每個懷有夢想的實踐者，踏出實現的第一步，最終建構出優秀的成品！

你也可以透過我們的官方網站了解更多細節: https://www.codecat.tw

### 專案導向

有別於多數社群以單一項目進行，程式貓社群採用專案導向！ 這樣的架構賦予每位成員發揮專長的空間，並且能夠在不同專案中學習到更多元的技能。 你可以根據自身的興趣為不同的項目做出貢獻，也可提出新的計畫讓成員一起幫你實現！

專案列表:

- 專題百科
- 科技週刊
- CPE Guide
- Takasagos war 台域戰策
- Next.js / React 讀書會
- Kitopia 貓托邦

你也可以在官網的[社群專案](https://www.codecat.tw/projects)頁面了解更多
