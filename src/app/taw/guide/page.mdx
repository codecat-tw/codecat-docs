# 遊戲教學

選擇你的所屬陣營，攻擊周圍的土地，為了團隊繼續攻城掠地。

佔領土地可以獲得資源，用來兌換道具增加進攻手段

佔領最多土地的聯盟可以獲得勝利。

## 常見問題

### 關於土地佔領

攻擊土地時，會減少和土地防禦力等值的體力，當土地防禦力被削減歸零時，便成功占領土地

### 關於中立土地

中立土地的防禦值為25點，占領後會變成自己的土地，並且防禦值變為5

### 關於已佔領土地

被佔領的土地初始血量為5，可透過消耗體力強化，單次最多可強化500點

### 關於資源獲取

攻擊中立土地可獲得50金幣，攻擊敵方土地則可獲得100金幣，金幣可至商店購買道具，增加攻擊與防禦的資源

### 關於體力恢復

十分鐘將會回復10點體力，如果急需體力也可在商店中購買恢復道具

### 關於康普頓立場

能夠阻止敵方進攻，堪比「防護罩」的道具。重複使用無法疊加時間，可以在攻防戰中發揮奇效
